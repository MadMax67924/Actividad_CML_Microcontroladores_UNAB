# Predicción de Temperatura con Random Forest

Este proyecto implementa un flujo de **Continuous Machine Learning (CML)** usando GitHub Actions para entrenar un modelo **Random Forest** que predice la temperatura máxima.

## Estructura del proyecto
- `train.py`: Script principal para entrenamiento.
- `requirements.txt`: Dependencias del proyecto.
- `.github/workflows/cml.yaml`: Pipeline para GitHub Actions.
- `data.csv`: Dataset de entrenamiento.

## Cómo ejecutar localmente
```bash
pip install -r requirements.txt
python train.py
```
